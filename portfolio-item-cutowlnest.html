Cutowl Nest

Web App

Experience at: Cutowl Srl
Role: Full Stack Web Developer
Date: 2025

Web-based, multi-player terminal game that challenged participants to outsmart an AI by crafting prompts and uncovering hidden codes.

My Role

I was responsible for implementing the game mechanics and building the web application. My work focused on creating the retro terminal interface, integrating OpenAI, and developing the backend features that powered registration, code submission, and the leaderboard.

Context & Goals

At the Apple Developer Academy’s Future Fair 2025, our team wanted to stand out in an environment filled with demos and pitches. The goal was to create an experience that was memorable, playful, and technically relevant, while also helping us connect with talented students. The game had to be easy to access, engaging to play, and lightweight enough to run in a web browser during a busy event.

Responsibilities

  • Implemented the terminal-style web interface with a typing effect and vaporwave aesthetic
  • Integrated OpenAI’s API to create a custom chatbot restricted to our curated knowledge base
  • Developed the game logic for file exploration, prompt interactions, and hidden code discovery
  • Built the backend using Supabase to handle user registration, authentication, and leaderboard tracking
  • Connected the scoring system so that submitted codes mapped to different point values
  • Tested the game flow to ensure smooth interactions across browsers and devices

Key Features Delivered

  • Animated boot sequence and command-line style interface
  • Custom GPT integration tied only to the game’s knowledge base
  • Gameplay loop where players explored files and extracted hidden codes
  • Live leaderboard that updated as players submitted valid codes
  • Secure backend system ensuring fair play and preventing duplicate submissions

Outcomes

  • Over 20 players registered and actively participated during the two-day event
  • The game drew steady traffic and kept participants engaged longer than traditional demos
  • The leaderboard helped identify standout players, which supported our recruitment goals
  • The project became an internal showcase of how to combine AI, gamification, and web development into an effective event tool

Personal Note

Working on Cutowl Terminal gave me the chance to combine several skills: frontend development, backend integration, and AI-powered interaction design. Beyond the technical aspects, I enjoyed seeing how something I helped build turned into an experience people lined up to play — a reminder of how impactful code can be when paired with a strong idea.
